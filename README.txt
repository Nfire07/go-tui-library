## Usage

```bash
go run main.go example.json
```

## Features

- **Layouts**: Column (default) and Flex layouts
- **Elements**: div, text, input, checkbox, button
- **Styling**: Colors, backgrounds, borders
- **Interactive**: Tab navigation, keyboard input

## Controls

- **Tab**: Navigate between focusable elements
- **Enter/Space**: Toggle checkboxes, activate buttons
- **Type**: Input text in focused input fields
- **Ctrl+C / Q**: Quit application
